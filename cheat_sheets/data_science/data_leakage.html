Data leakage

Information

Data leakage is when the training data contains information about the target, but similar data is not available.

The model will look accurate but the performance will be low on other data, e.g., test data.

There are two types of data leakage:

  • Target-leakage
  • train-test contamination
Usage

Target leakage occurs when your predictors include data that will not be available at the time you make predictions.

Train-test contamination occurs when there is overlapping data between training, validation, or test.

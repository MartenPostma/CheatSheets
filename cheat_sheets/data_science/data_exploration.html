Data exploration

Information

Exploring your data is an essential step in defining your model.

What are the features, i.e., columns?

What is the data type of each feature, i.e., continuous, categorical, ordinal?

Determine the follownig statistics about each feature (if possible depending on the data type):

  • number of non-missing values
  • number of missing values
  • the arithmic mean
  • the standard deviation
  • the minimum value
  • the maximum value
  • the percentiles: 25%, 50%, 75%

Show the top 5 and bottom data instances.

Usage

Pandas is the most used library to perform data exploration.

pandas.DataFrame.describe() can be used to determine the descriptive statistics.

df.head() can be used to show to the top 5 data instances.

df.tail() can be used to show the bottom 5 data instances.

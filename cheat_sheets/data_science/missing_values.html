Missing Values

Information

It occurs that not all data instances have values for all features:

  • a feature can only be applied to a subset of all data instances.
  • only a subset of all data instances have been annotated for that feature.

Three typical ways to treat missing values are:

  • drop columns with missing values
  • Imputation fill in one value, e.g., the mean of the column
  • Imputation extension Imputation, but with another column (boolean) to indicate whether there was a missing value.

Categorical variables

Information

Categorical variables are variables that take a limited number of values.

Cardinality is used to indicate the number of unique values for a categorical variable.

Three approaches to treat them are:

  • Drop categorical variables
  • Label encoding assign each unique value to a different integer. Please note that this creates artificial distance between the categories since the values are converted to integers.
  • One-hot encoding new columns are created for each unique value. The column indicates whether the value of variables applies to the data instance. Please note that One-hot encoding adds a large number of columns.
Usage

from sklearn.preprocessing import LabelEncoder can be used for Label Encoding.

from sklearn.preprocessing import OneHotEncoder can be used for One hot encoding.

SHAP

Information

SHapley Additive exPlanations (SHAP) shows insights into a model's single prediction.

SHAP compares the prediction made to the prediction when the feature value would be a baseline value.

Usage

import shap

explainer = shap.TreeExplainer(my_model)

shap_values = explainer.shap_values(data_for_prediction)

shap.initjs();shap.force_plot(explainer.expected_value[1], shap_values[1], data_for_prediction)

shap.summary_plot can be used for summary plots

shap.dependence_plot can be used for dependence contribution plots.

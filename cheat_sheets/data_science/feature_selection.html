Feature Selection

Information

An important step is to determine which feature will be selected.

The most common options are:

  • Univariate feature selectionFor each feature, measure how strongly the target depends on the feature using a statistical test

    • L1 regularization penalizes the absolute magnitude of the coefficients. As the strength of regularization is increased, features which are less important for predicting the target are set to 0.

Usage

from sklearn.feature_selection import SelectKBest can be used for Univariate Feature Selection.

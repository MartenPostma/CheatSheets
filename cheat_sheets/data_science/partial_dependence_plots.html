Partial dependence plots

Information

Partial dependence plots show how a feature affects predictions.

Partial dependence plots are calculated after the model has been fit.

One predictive feature is selected

The value of the one feature is repeatedly altered and predictions are made using this altered data.

The y-axis is the change in prediction compared to the baseline model.

Usage

from pdpbox import pdp;pdp.pdp_isolate can be used

pdp.pdp_interact can be used for 2D plots.

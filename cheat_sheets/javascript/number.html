Number

How to create
let thousand = 1000;
Useful methods

toString : cast number to string

Math.floor : rounds down

Math.ceil : rounds up

Math.round : rounds to the neirest integer

+ or Number : convert to integer

+"1"; // 1

Number("1"); // 1

Number("str"); // NaN

parseInt : parse integer from string

parseInt("1px"); // 1

parseInt("12.3"); // 12

parseFloat : parse float from string

parseFloat("12.3px"); // 12.3

parseInt("1px"); // 1

Pay attention to

float divisions : they can be imprecise and not as you expect them to be.

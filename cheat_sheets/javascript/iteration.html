Iteration

For

for (begin; condition; step) { loop body }

for (let i = 0; i < 3; i++) {alert(i)}

You can remove all three parts of declare them outside of the loop.

break used to break the loop

continue go to next iteration in the loop

Array

Array.foreach(function); run function for each element in array

Object

Loop over all keys in object

for (key in object) {
// executes the body for each key among object properties
}

Map

Map.keys() returns an iterable over keys

Map.values() returns an iterable over values

Map.entries() returns an iterable over [key, value]

Set

Set.keys() returns an iterable over keys

Set.values() returns an iterable over values

Set.entries() returns an iterable over [value, value]

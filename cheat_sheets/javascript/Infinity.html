Infinity

How to create

x = Infinity; // greater than everything

y = -Infinity; // smaller than everything

Useful methods

isFinite(value) converts to number. true for regular number, false for Nan|Infinity|-Infinity

isFinite(1); // true

isFinite(Infinity); // false

isFinite(-Infinity); // false

isNan converts to number and checks for NaN

isNaN(NaN); // true

isNaN('a'); // true

isNaN(1); // false

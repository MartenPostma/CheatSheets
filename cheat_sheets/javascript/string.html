How to create

Strings can be enclosed within either single quotes, double quotes or backticks:

let single = 'hello';

let double = "world";

backticks allow us to embed any expression and the string can span multiple lines.

let backticks = `${single} ${double}`; // hello world

Useful methods

String.length: length of string

String.toUpperCase(): change case to upper case

String.toLowerCase(): change case to upper case

String.includes(index, pos): substring check

String.startsWith(str): starts with

String.endsWith(str): ends with

String.trim(): removes spaces from the beginning and ending of the string

Accessing characters using index

let hello = 'hello';

hello[0] // h;

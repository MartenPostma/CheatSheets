Object

How to create

let user = new Object();

let user = {};

let user = { // an object
name: "John", // by key "name" store value "John"
age: 30 // by key "age" store value 30 }

Mulit-word attribute names must be quoted
let user = {
name: "John",
age: 30,
"likes birds": true // multiword property name must be quoted
};

If the name of the attribute and the value match, you can do this: let user = {
name, // same as name:name
age: 30
};

Useful methods

Object.attribute_name ; access attribute

Object["attribute name"] ; access attribute (useful for multiwords

delete Object.attribute_name ; delete attribute

Property existence:
undefined: Object.attribute_name_not_exists
boolean expression: attribute_name in Object

Console

Information

The console in Javascript and Node.js are similar.

Most used method is console.log() which prints the string you pass to it to the console.

String formatting can be done like this: console.log('My %s', 'cat')

console.clear() clears the console

console.count() keeps an internal dictionary how many times a certain string has been send to std with the console

console.time(label) and console.timeEnd(label) are used to show how long something takes to run.

STDOUT:console.log()

STDERR:console.error()

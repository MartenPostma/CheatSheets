callback

Information

A function that is to be executed after another function has finished executing.

Higher-order function: functions that take as argument or return another function.

Any function that is passed as an argument: callback function.

Usage

The first argument to a callback is reserverd for an error object.

The second argument to a callback is reserverd for any succesful response data.

The higher-order function is called first, e.g., fs.readFile('/foo.txt'

This function attempts to read the contents of the file.

The callback handles the output of this function call and has two arguments:

  • err contains error object in the case of an error.
  • data the succesful data response from the higher-order function.

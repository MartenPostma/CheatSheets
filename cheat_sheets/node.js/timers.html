Timers

Information

setTimeout delay function with at least x ms.

You can run a function with arguments:

setTimeout(myFunction, 2000, firstParam, secondParam)

setTimeout returns the timer id.

You can use clearTimeout(timer id) to remove the function call.

setInterval is used to run a function forever, at the specific time interval you specify (in milliseconds)

It returns the timer id. You can use clearTimeout(timer id) to stop it.

Running Node.js from the command line

Information

The default way of starting a Node.js application is to call from the command line:

node app.js

You can kill an application in the command line with ctrl-C

You can kill an application from within Node.js wih process.exit()

SIGKILL immediately kills the program and is similar to process.exit()

SIGTERM is used to gracefully end a program

You can send the SIGTERM signal from within a program using process.kill(process.pid, 'SIGTERM')

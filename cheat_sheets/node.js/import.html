Import

Information

You can import functionality from other Node.js files using

const library = require('./library')

This exposes the functionality in library.js

Any other object or variable defined in the file by default is private and not exposed to the outer world.

To expose functionality, you use the module.exports API.

This can be done in the following way:

1. Assign an object to module.exports.ATTRIBUTE_NAME

In file utils.js

let car = "tesla";

module.exports.car = car;

You can import this object in another file using:

let utils = require('./utils');

console.log(utils.car);

The console will show:

tesla

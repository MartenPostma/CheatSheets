Command line arguments

Information

You can pass any number of arguments when invoking a Node.js application:

node app.js joe peter kobe

There are two types of arguments:

Standalone: node.app.js joe

Key and value: node.app.js name=joe

You retrieve these command line arguments using process

process.argv contains

  • It exposes an argv property, which is an array that contains all the command line invocation arguments.
  • the first argument is the full path of the node command
  • the second argument is the path of the node application
  • all others are command line arguments

Parsing standalone arguments:

const args = process.argv.slice(2)

Parsing key=value arguments:

Call from the command line: node app.js --key=value --key1=value1

const args = require('minimist')(process.argv.slice(2))

args['name'] //joe
